<img width="1919" height="969" alt="image" src="https://github.com/user-attachments/assets/5aee1a1f-d4cb-4748-a7fc-2b36c5b5a792" /># Aplikasi Kasir (Versi 1) Sederhana Menggunakan Php + Mysql + Bootstrap

Dibuat dalam studi kasus Toko Modern.
Aplikasi ini benar benar cocok untuk pemula yang mau belajar PHP dan Mysql dari dasar.

Aplikasi Kasir adalah sistem Point of Sale (POS) sederhana berbasis web yang dibangun menggunakan PHP, MySQL, dan Bootstrap. Aplikasi ini dirancang khusus untuk membantu pemula mempelajari pemrograman web dengan studi kasus yang praktis dan menggunakan teknologi barcode untuk manajemen produk.
#Fitur Utama
1. Dashboard Informatif

Tampilan ringkasan statistik real-time
Total Barang (11 produk tersedia)
Total User (2 pengguna terdaftar)
Total Transaksi (8 transaksi tercatat)
Stok Menipis (monitoring produk yang perlu restock)
Visualisasi data dengan kartu berwarna yang menarik

2. Manajemen Barang

Pengelolaan produk menggunakan sistem barcode
CRUD (Create, Read, Update, Delete) produk
Monitoring stok barang
Notifikasi stok menipis

3. Manajemen User

Sistem multi-user
Role/hak akses pengguna
Manajemen pengguna (admin, kasir, dll)

4. Transaksi Penjualan

Pencatatan transaksi penjualan
Sistem kasir yang user-friendly
Riwayat transaksi

5. Diskon Barang

Pengaturan diskon produk
Manajemen promosi

6. Laporan Keuangan

Upload laporan keuangan
Monitoring performa bisnis
Analisis penjualan

#Teknologi yang Digunakan

Backend: PHP
Database: MySQL
Frontend: Bootstrap (responsive design)

#Target Pengguna

Pemula yang ingin belajar PHP dan MySQL
Pemilik toko/warung kecil
UMKM yang membutuhkan sistem kasir sederhana
Mahasiswa untuk project/tugas akhir

#Kelebihan

Sederhana dan mudah dipahami - cocok untuk pembelajaran
Interface user-friendly - mudah digunakan
Responsive design - dapat diakses dari berbagai device
Gratis dan open source - dapat dikembangkan sesuai kebutuhan
